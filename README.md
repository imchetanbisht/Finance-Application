#  FinTrack – Personal Finance Companion App

##  Overview

**FinTrack** is a modern and user-friendly personal finance mobile application built using Flutter.
It helps users track their daily expenses and income, understand spending patterns, and manage savings effectively.

This project focuses on clean UI, smooth user experience, and structured business logic, making it both practical and scalable.

---

## Features

### Dashboard

* Overview of total balance, income, and expenses
* Recent transactions preview
* Clean and minimal UI for quick insights

###  Transaction Management

* Add new transactions (Income / Expense)
* Edit existing transactions
* Delete transactions with swipe gesture
* Undo delete functionality
* Search transactions
* Filter by Income / Expense

###  Insights

* Visual representation of income vs expense
* Bar chart for better understanding
* Summary cards for financial overview

###  Savings Goal

* Track savings progress
* Circular progress indicator
* Motivational UI

### Smart UX Features

* Real-time search
* Filter chips
* Smooth animations
* Empty states handling

### Local Storage

* Data persists using SharedPreferences
* Automatically loads data on app start

### Splash Screen

* Lottie animation based splash screen
* Smooth app entry experience

## Tech Stack

* **Flutter (Dart)**
* **Provider** (State Management)
* **SharedPreferences** (Local Storage)
* **fl_chart** (Charts)
* **Lottie** (Animations)
* **UUID** (Unique IDs)
